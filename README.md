
# Wisely

This is a NextJS starter in Firebase Studio designed for personal and shared expense management.

## Project Structure

- `src/app`: Contains the application routes and logic (App Router).
- `src/components`: Reusable UI components (ShadCN, Layout, etc.).
- `src/firebase`: Firebase configuration and custom hooks for Firestore/Auth.
- `public/`: **(Recommended)** Place your static assets here (icons, images, manifest, etc.). Assets in this folder are served at the root path (e.g., `/icon.png`).

## Getting Started

To get started, take a look at `src/app/page.tsx`.

## PWA Support

Wisely is configured as a Progressive Web App. You can install it on your home screen from your browser. 
- **Android**: Use the "Add to Android Home Screen" button in your Profile settings.
- **iOS**: Use the Share menu in Safari and select "Add to Home Screen".
